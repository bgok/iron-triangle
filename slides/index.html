What is constraint programming

  • A programming paradigm where relationships between variables are defined
  • Variables are constrained to the defined relationships
  • Variables are not common primitives
  • Differs from imperative programming because a step or sequence of steps is not defined. Instead the properties of a solution are defined.

A Very Simple Example

Implementation

Variables

Constraints
